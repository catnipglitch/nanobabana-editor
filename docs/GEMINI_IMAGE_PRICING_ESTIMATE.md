# Gemini 3 Pro Image Preview 料金試算

Gemini 3 Pro Image Preview モデルを使用して画像生成を行う際のコスト試算表です。
公式の [Gemini API Pricing](https://ai.google.dev/gemini-api/docs/pricing) に基づく推定値です。

> **注意**: 通貨レートは **1 USD = 150 JPY** で計算しています。実際の請求額は為替レートやGoogle Cloudの課金ポリシーにより変動します。

## 1. 基本料金単価 (Unit Pricing)

| モデル | 種別 | 100万トークンあたり | 1トークン単価 (USD) |
| :--- | :--- | :---: | :---: |
| **gemini-3-pro-image-preview** | 入力トークン（テキスト） | $2.00 | $0.0000020 |
| | 入力トークン（画像） | $2.00 | $0.0000020 |
| | 出力トークン（テキスト/思考）| $12.00 | $0.0000120 |
| | 出力トークン（画像） | $120.00 | $0.0001200 |

## 2. 画像関連の固定トークン消費量

Gemini 3 では、画像の入力・出力に対して固定のトークン数が割り当てられます。

| 種別 | 解像度 / サイズ | 消費トークン数 |
| :--- | :--- | :---: |
| **入力画像** | - | 560 tokens |
| **出力画像** | 1,024×1,024 (1K, 2K) | 1,120 tokens |
| | 1,024×1,024 (4K) | 2,000 tokens |
